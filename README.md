# Getting Started

Clone this repo

```
git clone "Repo Link"
```

Change directory to BD-Tour-Backend

```
cd BD-Tour-Backend
```

Install the dependencies

```
npm install
```

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:8000](http://localhost:8000) with your browser to see the result.
